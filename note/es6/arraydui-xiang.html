    Array对象

    党洁 11/16/2021 js 函数

    # 一、定义

    • 字面量
    let arr = [1, 2, 3, { name: "dangjie" }];
    • 利用 Array 构造函数
    //单个正整数参数，表示返回的新数组的长度
let arr = new Array(1);

//非数值（比如字符串、布尔值、对象等）作为参数，
// 则该参数是返回的新数组的成员
new Array("abc", 1); // ['abc',1]

    # 二、属性

    length:数组长度，并且是动态的，可写，会影响数组。

    # 运算符

    检查某个键名是否存在的运算符 in，适用于对象，也适用于数组。

    var arr = [];
arr[100] = "a";

100 in arr; // true
1 in arr; // false

    # 三、方法

    # 静态方法

    1. Array.isArray( ) ： 判断一个值是否为数组。弥补 typeof 运算符的不足。
    var arr = [1, 2, 3];

typeof arr; // "object"
Array.isArray(arr); // true
    1. Array.from( ) : 将类数组(可迭代对象)浅拷贝成数组实例。
    Array.from("string"); // [ 's', 't', 'r', 'i', 'n', 'g' ]

    如何将类数组转化成数组？

    1. 静态方法 Array.from( )
    2. 数组实例的 slice 方法可以将“类似数组的对象”变成真正的数组。
    //通过call(),将数组的方法放到类数组对象上，即可使用slice()
var arr = Array.prototype.slice.call("string"); // [ 's', 't', 'r', 'i', 'n', 'g' ]
    1. 扩展操作符...
    const items = new Set([1, 2, 3, 4, 5, 5, 5, 5]);
[...items]; //[1,2,3,4,5]
    1. 循环遍历类数组对象，push 到新创建的数组对象里
    let str = "string";
let arr = [];
for (var i = 0; i < str.length; i++) {
  arr.push(str[i]);
}
console.log(arr);
    1. Array.of( ): 将一组参数转化成数组。

    # 实例方法

    # 1.添加和删除（都会改变原数组）

    • push( )： 在数组的末端添加元素，并返回添加新元素后的数组长度。

      pop( ) ： 删除数组的最后一个元素，并返回该元素。

      push 和 pop 结合使用，就构成了“后进先出”的栈结构（stack）,在栈顶操作。

    var arr = [];
arr.push(1, 2, {}); // 3
arr.pop(); //{}
arr; //[1,2]

    • unshift( )：在数组的首部添加元素，并返回添加新元素后的数组长度。

      shift( )：删除数组第一个元素，并返回该元素。

      push 和 shift 结合使用，就构成了“先进先出”的队列结构。

    var arr = [];
arr.unshift("a", "b", {}); //3
arr.shift(); // 'a'
arr; //['b',{}]

    # 2. join()

    join()方法以指定参数作为分隔符，将所有数组成员连接为一个字符串返回。如果不提供参数，默认用逗号分隔。

    var a = [1, 2, 3, 4];

a.join(" | "); // "1 | 2 | 3 | 4"
a.join(); // "1,2,3,4"

    # 3. concat()

    concat()方法用于多个数组合并，返回新数组，原数组不变。

    ["hello"].concat(["world"], ["!"]); // ["hello", "world", "!"]

    # 4. slice()

    slice(start,end)用于提取目标数组的一部分，返回一个新数组，原数组不变。

    slice()没有参数，实际上等于返回一个原数组的浅拷贝。

    var a = ["a", "b", "c"];

a.slice(1); // ["b", "c"]
a.slice(1, 2); // ["b"]
a.slice(2, 6); // ["c"]
a.slice(); // ["a", "b", "c"]

    # 5. splice()(会改变原数组)

    splice(start, count, addElement1, addElement2, ...)方法用于删除原数组的一部分成员，并可以在删除的位置添加新的数组成员，返回值是被删除的元素。会改变原数组。

    var a = ["a", "b", "c", "d", "e", "f"];
a.splice(4, 2, 1, 2); // ["e", "f"]
a; // ["a", "b", "c", "d", 1, 2]

    # 6. reverse()(会改变原数组)

    reverse()颠倒排列数组元素，返回改变后的数组。

    var a = ["a", "b", "c"];

a.reverse(); // ["c", "b", "a"]
a; // ["c", "b", "a"]

    # 7.sort()(会改变原数组)

    sort()对数组成员进行排序，默认是按照字典顺序排序。可传入函数作为参数，自定义排序。原数组将被改变。

    [1, 4, 2, 6, 0, 6, 2, 6].sort((a, b) => a - b);

    # 8.查找元素

    • includes():返回布尔值
    [1, 2, 3, 4].includes(1); //true
    • indexOf()、lastIndexOf():返回第一次出现的索引位置，若没有返回-1.
    • find()、findIndex()方法：使用了断言函数。find()返回第一个匹配的元素，findIndex()返回第一个匹配元素的索引。
    const people = [
{
name: "Matt", age: 27
},
{
name: "Nicholas", age: 29
}
];

people.find((element, index, array) => element.age < 28);
// {name: "Matt", age: 27}

(people.findIndex((element, index, array) => element.age < 25);
// -1

    # 9. 迭代器方法

    • keys( )：返回数组索引的迭代器。
    • values( ): 返回数组元素的迭代器。
    • entries( ): 返回索引/值对的迭代器。
    const a = ["foo", "bar", "baz", "qux"];

// 迭代器通过Array.from()直接转换为数组实例
const aKeys = Array.from(a.keys()); //[ 0, 1, 2, 3 ]
const aValues = Array.from(a.values()); //[ 'foo', 'bar', 'baz', 'qux' ]
const aEntries = Array.from(a.entries()); //[ [ 0, 'foo' ], [ 1, 'bar' ], [ 2, 'baz' ], [ 3, 'qux' ] ]

    # 10. 迭代方法

    • map( )：将数组的所有成员依次传入参数函数，然后把每一次的执行结果组成一个新数组返回。
    var arr = ["a", "b", "c"];

// 第二个参数，用来绑定函数内部this变量。
[1, 2].map(function (elem, index, arr) {
  return this[elem];
}, arr);
// ['b', 'c']
    • forEach( ): 和 map()类似，无返回值。

      forEach 和 map 区别：map 会返回新数组，可以进行链式操作；forEach 无返回值。

    forEach 和 for 的区别：for 可以 break 退出循环，forEach 和 map 不能。 :::

    [1, 2, 3].forEach(function (elem, index, arr) {
  elem * index;
});
    • filter( ): 用于过滤数组成员，满足条件的成员组成一个新数组返回。
    [1, 2, 3, 4, 5].filter(function (elem) {
  return elem > 3;
});
// [4, 5]
    • some( ): 只要一个成员的返回值是 true，则整个 some 方法的返回值就是 true
    var arr = [1, 2, 3, 4, 5];
arr.some(function (elem, index, arr) {
  return elem >= 3;
});
// true
    • every( ): 所有成员的返回值都是 true，整个 every 方法才返回 true.
    var arr = [1, 2, 3, 4, 5];
arr.every(function (elem, index, arr) {
  return elem >= 3;
});
// false

    # 11. 复制和填充

    • fill(定值，开始位置，结束位置 )：使用给定值，填充一个数组。
    ["a", "b", "c"].fill(7, 1, 2); //['a', 7, 'c']
    • copyWithin(target,开始位置，结束位置): 在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），然后返回当前数组。改变原数组。
    [1, 2, 3, 4, 5].copyWithin(0, 3);
// [4, 5, 3, 4, 5]

    # 12. 归并方法

    • reduce(归并函数，，归并起点初始值)、reduceRight( )：依次处理数组的每个成员，最终累计为一个值.
    let number = [1, 2, 3, 4, 5];
values.reduce((prev, cur, index, array) => prev + cur, 1); //16

    # 四、Set 集合（可迭代对象）

    # 1. 定义

    const s = new Set([1, 2, 3, 4, 5, 5, 5, 5]);
s.size; //5

    # 2. 属性和方法

    • size：返回 Set 实例的成员总数。
    • add(value)：添加某个值，返回 Set 结构本身。
    • delete(value)：删除某个值，返回一个布尔值，表示删除是否成功。
    • has(value)：返回一个布尔值，表示该值是否为 Set 的成员。
    • clear()：清除所有成员，没有返回值。
    s.add(1).add(2).add(2);
// 注意2被加入了两次
s.size; // 2
s.has(1); // true
s.has(2); // true
s.has(3); // false

s.delete(2);
s.has(2); // false

    # 五、Map 映射（可迭代对象）

    # 定义

    给 Map 构造函数传入一个可迭代对象，需要包含键/值对数组。

    const m1 = new Map([
  ["key1", "val1"],
  ["key2", "val2"],
  ["key3", "val3"],
]);
m1.size //3

    # 属性和方法

    和set类似，多一个get()方法

    m1.get('key1') //val1
