    函数

    党洁 11/16/2021 js 函数

    # 一、概述

    # 1、 函数定义

    JavaScript 有三种声明函数的方法。

    1. 函数声明式:function 命令声明，整个函数（包括函数体）会被提升到代码头部。
    function print(s) {
  console.log(s);
}
    1. 函数表达式：采用变量赋值的写法，将一个匿名函数赋值给变量。
    var print = function (s) {
  console.log(s);
};
//箭头函数没有arguments、super和new.target,this的指向也和普通函数不同。
var print = () => {
  console.log(s);
};
    1. Function 构造函数
    var add = new Function(
  "x",
  "y",
  "return x + y" //最后一个参数会被当做函数体。
);

    # 2、return 语句

    return 不是必需的。如果没有的话，该函数就不返回任何值，或者说返回 undefined。

    return 的作用有两点：

    • 函数的返回值。
    • 立即停止执行函数并退出。

    # 3、函数声明提升

    • 函数声明式：整个函数会被提升到代码头部。
    f(); //111

function f() {
  console.log("111");
}
    • 函数表达式、Function 构造函数：和变量赋值一样，var 会变量提升，但函数体仍留在原地；let 不会提升。
    f();
var f = function () {
  console.log("111");
};
// TypeError: undefined is not a function

    # 二、函数属性和方法

    # 属性

    • name：函数名称或变量名。
    • length：命名参数（形参）个数。

    # 方法

    • apply(this 的指向，可迭代对象):
    function getAge() {
  return this.birth;
}

var xiaoming = {
  name: "小明",
  birth: 1990,
  age: getAge,
};
xiaoming.age(); // 25
getAge.apply(xiaoming, []); //25
    • call(this 的指向，单个参数)。
    • bind():bind()方法会创建一个新的函数实例，其 this 值会被绑定到传给 bind()的对象。
    color = "red";
var o = {
  color: "blue",
};

function sayColor() {
  console.log(this.color);
}
let objectSayColor = sayColor.bind(o);
objectSayColor(); // blue

    # 三、理解参数

    # 1、arguments

    # 定义

    arguments 对象是一个类数组对象，接收函数的实参，不反映参数的默认值。

    arguments 对象可以跟命名参数一起使用，但严格模式下，不允许重写 arguments。

    function doAdd(num1, num2) {
  "use strict"; // 开启严格模式
  arguments[1] = 10;
  console.log(arguments[0] + num2);
}
doAdd(1, 2); // 3

    # 属性

    • length：通过 arguments 对象的 length 属性，可以判断函数调用时到底带几个参数。
    • callee：返回它所对应的原函数，可解决递归的耦合问题。严格模式下禁用。
    function factorial(num) {
  if (num <= 1) {
    return 1;
  } else {
    return num * arguments.callee(num - 1);
  }
}

    # 2、默认参数

    在函数定义中的参数后面用=就可以为参数赋一个默认值：

    给多个参数定义默认值实际上跟使用 let 关键字顺序声明变量一样。后定义默认值的参数可以引用先定义的参数

    function makeKing(name = "Henry", numerals = name) {
  return `King ${name} ${numerals}`;
}
console.log(makeKing()); // King Henry Henry

    默认参数值并不限于原始值或对象类型，也可以使用调用函数返回的值：

    let romanNumerals = ["I", "II", "III", "IV", "V", "VI"];
let ordinality = 0;
function getNumerals() {
  // 每次调用后递增
  return romanNumerals[ordinality++];
}
function makeKing(name = "Henry", numerals = getNumerals()) {
  return `King ${name} ${numerals}`;
}

console.log(makeKing()); // 'King Henry I'
console.log(makeKing()); // 'King Henry II'
console.log(makeKing()); // 'King Henry III'

    # 3、 参数扩展与收集

    ES6 新增了扩展操作符（...），使用它可以非常简洁地操作和组合集合数据。

    # 扩展参数

    对可迭代对象应用扩展操作符，可以将可迭代对象拆分，将迭代返回的每个值单独传入。

    let values = [1, 2, 3, 4];

function getSum() {
  let sum = 0;
  for (let i = 0; i < arguments.length; ++i) {
    sum += arguments[i];
  }
  return sum;
}
//不妨碍在其前面或后面再传其他的值。
getSum(...values, -1); //9

    # 收集参数

    在构思函数定义时，可以使用扩展操作符把不同长度的独立参数组合为一个数组。只能把它作为最后一个参数：

    let getSum = (...values) => {
  return values.reduce((x, y) => x + y, 0);
};
console.log(getSum(1, 2, 3)); // 6

    # 四、函数内部对象

    # 1、arguments 对象（上面已有）

    # 2、this 对象

    this 指的是该属性或方法“当前”所在对象。

    • 在标准函数中，this 引用的是函数被调用时的上下文对象。
    var color = "red";
let o = {
  color: "blue",
};
function sayColor() {
  console.log(this.color);
}
sayColor(); //red
o.sayColor = sayColor;
o.sayColor(); //blue

    内部函数不能直接访问到外部函数的 this 和 arguments 对象。

    但可以将 this 对象保存至另一个变量，内部函数即可访问到。

    birth = 1997;
var obj = {
  birth: 1990,
  getAge: function () {
    var b = this.birth; // 1990
    var fn = function () {
      return this.birth; // this指向window或undefined，1997
      return b; // 1990
    };
    return fn();
  },
};
    • 在箭头函数中，this 引用的是定义箭头函数的上下文。
    birth = 1997;
var obj = {
  birth: 1990,
  getAge: function () {
    var b = this.birth; // 1990
    var fn = () => {
      return this.birth; //this总是指向词法作用域，也就是外层调用者obj。1990
    };
    return fn();
  },
};
    • 在构造函数中，this 指的是实例对象。

    # 3、caller 对象

    这个属性引用的是调用当前函数的函数。

    function outer() {
  inner();
}

function inner() {
  console.log(inner.caller);
}
outer(); // outer

    # 4、new.target

    如果函数是正常调用的，则 new.target 的值是 undefined；如果是使用 new 关键字调用的，则 new.target 将引用被调用的构造函数。

    # 五、函数作用域链、垃圾回收及闭包

    # 1、 作用域链

    JavaScript的函数在查找变量时从自身函数定义开始，从“内”向“外”查找。直至全局作用域。

    # 2、 垃圾回收机制

    # 原理

    • 确定哪个变量不会再使用，然后释放它占用的内存。
    • 垃圾回收程序每隔一定时间就会自动运行。

    # 方法

    • 标记清除：当变量离开执行环境后（函数执行完毕），标记为“离开环境”；去除掉全局变量及闭包产生的特殊变量，剩下的是要删除的变量。
    • 计数清除：当声明一个变量并将一个引用类型赋值给该变量时该值引用次数加1，当这个变量指向其他一个时该值的引用次数便-1。当该值引用次数为0时，则说明没有办法再访问这个值了，被视为准备回收的对象. 计数清除方法可能导致循环引用，类似死锁，导致内存泄露。
    function problem() {
    var objA = new Object(); //1
    var objB = new Object(); //1
    objA.someOtherObject = objB;//2
    objB.anotherObject = objA;  //2
}

    # 3、闭包

    由于函数外部无法直接访问函数内部变量，我们可以在函数内部定义一个函数，并返回该函数，即可在外部访问到。

    function f1() {
  var n = 999;
  function f2() {
    console.log(n); //内部函数可以访问到外部函数的n
  }
  return f2; //返回内部函数
}

var result = f1(); 
result(); // 999

    # 闭包的作用

    • 可以读取其他函数内部的变量。
    • 让这些变量始终保持在内存中，造成内存泄露。
    function createIncrementor(start) {
  var n = 999
  return function () {
    var m = 1000
    return start++;    
  };
}
var inc = createIncrementor(5); //createIncrementor()函数执行完毕，它的局部活动对象应该被销毁（变量n），由于内部匿名函数在引用它的start变量，所以无法销毁start。

inc() // 5 inc()执行完，销毁m；start一直留存在内存中，造成内存泄露；除非解除对匿名函数的引用，才可以释放内存。
inc() // 6
inc() // 7
inc = null
    • 封装对象的私有属性和私有方法。

    # 4、立即调用的函数表达式

    不必为函数命名，避免了污染全局变量。

    (function () {
  var tmp = newData;
  processData(tmp);
  storeData(tmp);
}());

    # 六、回调函数

    回调函数是异步操作最基本的方法。

    回调函数就是一个被作为参数传递的函数。

    第一个参数应该是一个函数，第二个参数应该是要传给这个函数的值。

    function f1(callback,someArguments){
  return callback(someArguments)
}
function f2(){
  //
}
f1(f2,1)

    # 七、eval 命令

    eval命令接受一个字符串作为参数，并将这个字符串当作语句执行。

    eval('var a = 1;');
a // 1
